## 🌍 What We Build

VIT Network is the intelligence infrastructure layer for the African digital economy. We provide verifiable, trustless intelligence signals that power the next generation of African finance, sports, governance, and commerce.

| Product | Status | Description |
|---------|--------|-------------|
| 🏆 **Sports Intelligence** | ✅ Live | Real-time AI predictions across major football leagues with CLV tracking |
| 🤖 **Agent Swarm** | ✅ Live | 22 autonomous AI agents — market scouts, fraud reviewers, oracle sentinels |
| 💳 **Wallet & Payments** | ✅ Live | Multi-currency (NGN/USD/USDT/VITCoin) with Paystack & Stripe |
| ⛓️ **Blockchain Oracle** | ✅ Live | Gasless Base L2 transactions, VITCoin economy, W3C DID identity |
| 🏪 **Signal Marketplace** | ✅ Live | P2P intelligence trading with staking & slashing |
| 🌌 **Tachyon Fabric** | 🔶 Beta | Swarm storage with EEC erasure coding |
| 🗳️ **Elections & Policy** | 🔶 Beta | Electoral sentiment engine & policy simulation |
| 🎓 **VIT Academy** | ✅ Live | Gamified learning with XP, streaks & on-chain credentials |

## 🛠️ Technology Stack

**Backend**: Python 3.11 · FastAPI · SQLAlchemy · Alembic · Redis
**Frontend**: React 19 · Vite 6 · TypeScript · Tailwind CSS v4 · Radix UI
**AI/ML**: Scikit-learn · XGBoost · PyTorch · Statsmodels · 13-model ensemble
**Blockchain**: Solidity 0.8 · Foundry · Web3.py · Viem · Base L2
**Infrastructure**: Google Cloud Run · Docker · PostgreSQL · Redis
