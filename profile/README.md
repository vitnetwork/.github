## Migration Phases

VIT Chain is undergoing phased hardening before full settlement independence.
Real user funds do **not** move to VIT Chain until Phase 1 and Phase 2 gates are confirmed complete.

| Phase | Status | Gate Criteria |
|---|---|---|
| **1 — Consensus Hardening** | 🔄 In Progress | Validator bootstrapping · Slashing enforced in code · Idempotent genesis · Auto chain-state snapshots · Block explorer live |
| **2 — Security Review** | ⏳ Pending | Adversarial Sybil testing · Double-spend scenarios · P2P stress test · Red-team complete |
| **3 — Node Economy** | ⏳ Pending | Google Drive contributor onboarding · Node daemon · Reward economics at small scale |
| **4 — Settlement Migration** | 🔒 Gated | Real funds only after Phase 1 & 2 complete · Bridge/dual-write first |
| **5 — Full Independence** | 🔒 Gated | Retire external L2 dependency after Phase 4 verified stable |
